# SAMPLE CHOPPAH! 🚁🔪💿

> **The sample chopper built for scratch DJs, by scratch heads.**
> 
> *"I need your breaks, your stabs, and your vocal chops."* — The Scratchinator 🤖

## 🎯 What It Does

This ain't a DAW. This ain't a beat maker. This is a **sample choppah** — purpose-built for scratch DJs who need to build scratch sentences and sample tracks fast.

| Feature | The Deal |
|---------|----------|
| 🔗 **YouTube** | Paste a link, the Choppah resolves it. No downloads needed upfront. |
| 🔊 **SoundCloud** | Drop a SoundCloud link, same deal. Stream it, chop it, export it. |
| 📁 **Local files** | Got your own crate? Upload WAV, MP3, FLAC, OGG, and more. |
| 🔪 **Visual chopping** | Click & drag on the stream bar to select regions. See what you're cutting. |
| 🔬 **Precision overlay** | Click a region to zoom in. Drag handles or type exact timestamps. Surgical. |
| 🎨 **Per-region colors** | Every chop gets its own color so you never lose track of what's what. |
| 🚫 **Overlap prevention** | Regions can't bleed into each other. Clean cuts only. |
| ↩️ **Undo / Redo** | `Ctrl+Z` / `Ctrl+Y`. Chop fearlessly. |
| 📋 **Export queue** | Drag to reorder your chops. The timeline strip shows you the final sequence. |
| ✅ **Ready gating** | Only regions marked ready (✓) get exported. Stage your cuts. |
| ⏱️ **Adjustable gap** | Set the silence between chops: 0–10 seconds. Dial in your spacing. |
| 🔊 **Export preview** | Hear your scratch track before you download. Preview, tweak, export. |
| 💾 **WAV / MP3 / FLAC** | Export the whole scratch track as one file. |

## ⚡ Quick Start

### Prerequisites

You need **Python 3.10+** and **ffmpeg** on your system.

```bash
# macOS
brew install ffmpeg

# Ubuntu / Debian
sudo apt install ffmpeg

# Windows
winget install ffmpeg
# or grab it from https://ffmpeg.org/download.html
```

### Fire It Up

```bash
# Clone the repo
git clone https://github.com/Jofralso/sample-choppah.git
cd sample-choppah

# Set up (Linux/macOS)
bash scripts/setup.sh

# Set up (Windows)
scripts\setup.bat

# Or do it manually:
python3 -m venv .venv
source .venv/bin/activate      # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python app.py

# → http://localhost:5555
```

Open your browser. Drop a link. Start choppin'.

## 🎛️ How To Use It (The 30-Second Version)

1. **Paste a YouTube or SoundCloud link** (or search by name, or upload a file)
2. **Drag across the stream bar** to chop a region
3. **Click a region** to open the precision overlay and fine-tune
4. **Hit ✓** to mark a region as ready
5. **Drag regions** in the export queue to set the order
6. **Set the gap** between chops (0–10s) — default 1 second
7. **Pick your format** (WAV/MP3/FLAC) and smash **EXPORT**
8. **Preview** the result, then download

Your scratch track drops as a single file — every chop in order with your chosen gap spacing. Load it up and get busy.
